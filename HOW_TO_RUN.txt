📋 ШАГ 1: ПОЛУЧИТЕ ТОКЕНЫ
─────────────────────────────────────────────────────────────

1. Telegram Bot Token:
   • Откройте @BotFather в Telegram
   • Отправьте /newbot
   • Следуйте инструкциям
   • Скопируйте полученный токен

2. OpenAI API Key:
   • Зарегистрируйтесь на https://platform.openai.com
   • Перейдите в API Keys
   • Создайте новый ключ
   • Скопируйте его


📋 ШАГ 2: НАСТРОЙКА ПРОЕКТА
─────────────────────────────────────────────────────────────

1. Откройте терминал в папке iiSridhar_telegram

2. Скопируйте файл конфигурации:
   
   cp .env.example .env

3. Откройте .env в текстовом редакторе:
   
   nano .env
   # или
   code .env

4. Вставьте ваши токены:

   BOT_TOKEN=ваш_telegram_bot_token_сюда
   OPENAI_API_KEY=ваш_openai_api_key_сюда

5. Сохраните файл (Ctrl+O, Enter, Ctrl+X в nano)


📋 ШАГ 3: ЗАПУСК БОТА
─────────────────────────────────────────────────────────────

ВАРИАНТ A - Простой запуск (рекомендуется для первого раза):

   ./start.sh

   Если ошибка "Permission denied", выполните:
   chmod +x start.sh
   ./start.sh


ВАРИАНТ B - Напрямую через npm:

   npm start


ВАРИАНТ C - Режим разработки (с авто-перезагрузкой):

   npm run dev


📋 ШАГ 4: ЗАПУСК WEB APP (опционально)
─────────────────────────────────────────────────────────────

В НОВОМ окне терминала:

   ./start_webapp.sh
   # или
   npm run webapp

Web App будет доступен на http://localhost:3000


📋 ШАГ 5: ТЕСТИРОВАНИЕ
─────────────────────────────────────────────────────────────

1. Откройте Telegram
2. Найдите вашего бота (по username из BotFather)
3. Нажмите START или отправьте /start
4. Начните диалог! 💬

❌ Бот не запускается:
   → Проверьте .env файл
   → Убедитесь, что BOT_TOKEN заполнен
   → Проверьте интернет соединение

❌ AI не отвечает:
   → Проверьте OPENAI_API_KEY
   → Убедитесь, что на аккаунте есть баланс
   → Попробуйте перезапустить бота

❌ Ошибка базы данных:
   → Удалите папку data/
   → Перезапустите бота
   → Папка создастся автоматически
